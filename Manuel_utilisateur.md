# Campus Zen

Manuel utilisateur

Application mobile Campus Zen.

## Prérequis d'utilisation

- Une connexion Internet est nécessaire pour charger les données.
- Un compte Campus Zen est nécessaire pour se connecter.
- La localisation est conseillée pour l'onglet Maps, mais elle n'est pas obligatoire.

## Démarrage rapide

Si vous voulez aller droit au but :

1. Connectez-vous.
2. Allez dans Accueil, puis Questionnaire.
3. Répondez aux questions, puis validez.
4. Revenez sur Accueil, puis ouvrez Mon état.
5. Depuis Mon état, ouvrez la carte des professionnels si besoin.

## Prise en main

### Se connecter

1. Ouvrez l'application.
2. Sur l'écran Connexion, saisissez votre email et votre mot de passe.
3. Appuyez sur le bouton de connexion.
4. Une fois connecté, vous arrivez sur l'application avec les onglets en bas.

Si un message d'erreur apparaît :

- Vérifiez l'email et le mot de passe.
- Vérifiez que le téléphone a accès à Internet.
- Réessayez plus tard si le serveur ne répond pas.

### Créer un compte

1. Depuis l'écran Connexion, appuyez sur Inscription.
2. Remplissez les champs demandés.
3. Validez.
4. Après inscription, vous revenez à la connexion, puis vous pouvez vous connecter.

Si vous voyez un message du type « Email déjà utilisé » :

- Essayez un autre email.
- Ou connectez-vous avec le compte déjà existant.

### Se déconnecter

1. Allez dans l'onglet Compte.
2. Appuyez sur Se déconnecter.

### Changer la langue

1. Allez dans l'onglet Compte.
2. Dans la carte Langue, choisissez la langue dans le sélecteur.
3. Les textes principaux changent automatiquement.

## Navigation générale

En bas de l'écran, vous avez 4 onglets :

- Accueil
- Maps
- Ressources
- Compte

Le bouton retour du téléphone (ou la flèche de retour dans l'entête) sert à revenir à l'écran précédent.

Structure globale des écrans :

```
Auth
- Connexion
- Inscription

App
- Accueil
  - Questionnaire
  - Questions
  - Mon état
- Maps
  - Fiche professionnel
- Ressources
- Compte
```

Conseils :

- Si un écran reste sur « Chargement… » trop longtemps, revenez en arrière puis réouvrez l'onglet.
- Si une liste est vide, vérifiez le filtre et la recherche.
- Sur Maps, dézoomer aide souvent à retrouver des marqueurs.

## Accueil

L'onglet Accueil sert de point de départ.

Vous y trouvez surtout :

- Un accès direct au questionnaire.
- Un accès direct à Mon état quand vous avez déjà un statut.

### Lancer un questionnaire

1. Depuis Accueil, appuyez sur Questionnaire.
2. Choisissez un questionnaire dans la liste.
3. Répondez aux questions une par une.
4. Validez à la fin.

Après validation, un écran de succès s'affiche, puis vous pouvez consulter votre état.

### Consulter Mon état

1. Depuis Accueil, appuyez sur Mon état.

Le bouton peut être désactivé si aucun statut n'a encore été enregistré.

## Questionnaires

### Choisir un questionnaire

Sur l'écran « Questionnaires disponibles » :

- Touchez un questionnaire pour démarrer.
- Si la liste ne charge pas, utilisez Réessayer.

### Répondre aux questions

Pendant le questionnaire :

- Lisez la question.
- Choisissez une réponse.
- Continuez jusqu'à la dernière question.

À la fin, l'application envoie vos réponses.

Si vous êtes bloqué :

- Vérifiez que vous avez bien choisi une réponse.
- Vérifiez votre connexion Internet.
- Si une erreur persiste, revenez en arrière puis relancez le questionnaire.

### Écran de succès

Quand l'envoi réussit :

- Un écran de succès apparaît.
- Vous pouvez appuyer sur Consulter Mon état.

## Mon état

L'écran Mon état affiche votre dernier statut.

Vous pouvez y voir :

- Une image associée au climat.
- Un message motivant lié à ce climat.

Boutons disponibles :

- Consulter la carte des professionnels.
- Voir mes progrès.

Note :

- Selon la version, le bouton « Voir mes progrès » peut être visible sans action associée.

## Maps

L'onglet Maps sert à trouver des professionnels sur une carte et via une liste.

### Autoriser la localisation

Au premier lancement, la carte peut demander une permission.

- Si vous acceptez, l'application tente de se centrer sur votre position.
- Si vous refusez, une région par défaut est utilisée.

Vous pouvez activer la localisation plus tard dans les paramètres du téléphone.

### Utiliser la carte

Sur la carte :

- Zoomez et déplacez la carte.
- Touchez un marqueur pour sélectionner un professionnel.
- La liste en bas se met à jour selon la zone visible.

Si vous ne voyez pas le bon professionnel :

- Dézoomez un peu pour élargir la zone.
- Déplacez la carte vers une autre région.

### Utiliser la liste en bas

Une liste de professionnels est affichée dans un panneau en bas.

- Faites glisser le panneau vers le haut ou le bas pour l'ouvrir ou le fermer.
- Touchez une carte de professionnel pour centrer la carte sur lui.
- Touchez Détails pour ouvrir la fiche du professionnel.

### Fiche professionnel

Sur la fiche, vous pouvez :

- Appeler si un numéro est disponible.
- Envoyer un email si une adresse est disponible.
- Ouvrir l'application de cartographie selon votre plateforme.

## Ressources

L'onglet Ressources affiche des contenus utiles.

### Rechercher une ressource

1. Utilisez la barre « Rechercher une ressource… ».
2. La liste se filtre par titre et description.
3. Appuyez sur la croix pour effacer la recherche.

### Filtrer par type

Des boutons de filtre sont disponibles (exemples) :

- Tout
- Article
- Vidéo
- Podcast
- Livre
- Site web
- Documentaire
- Film
- Formation

### Ouvrir une ressource

1. Choisissez une ressource dans la liste.
2. Appuyez sur Voir la ressource.
3. Un navigateur (ou une page web) s'ouvre avec le lien.

Si aucun résultat ne s'affiche :

- Si aucune ressource ne correspond à votre recherche, un message l'indique.

## Compte

L'onglet Compte regroupe vos informations et des réglages rapides.

Vous y retrouvez notamment :

- Votre email.
- Votre identifiant.
- Votre rôle.
- Le sélecteur de langue.
- Le bouton de déconnexion.

## Fonctions avancées

Certains écrans existent pour gérer le contenu côté administration :

- Liste des questionnaires.
- Gestion des questions d'un questionnaire.
- Gestion des réponses d'une question.

Ces écrans ne sont utilisables que si votre version de l'application les propose et si votre compte a les droits nécessaires.

## Dépannage

### Je ne vois aucun contenu

- Vérifiez la connexion Internet.
- Réessayez plus tard si l'API est en maintenance.
- Relancez l'application.

### Les questionnaires ne chargent pas

- Utilisez le bouton Réessayer (s'il est présent).
- Vérifiez la connexion Internet.
- Déconnectez-vous puis reconnectez-vous si le problème persiste.

### Je ne peux pas ouvrir un lien de ressource

- Vérifiez la connexion Internet.
- Vérifiez que le lien commence par http ou https.
- Essayez depuis un autre réseau.

### La carte ne se centre pas sur moi

- Vérifiez la permission de localisation.
- Vérifiez que le GPS est activé.
- Attendez quelques secondes si le téléphone recherche la position.

### Je n'arrive pas à me connecter

- Vérifiez vos identifiants.
- Vérifiez la connexion.
- Si le problème persiste, demandez à un administrateur de vérifier votre compte.

## Questions fréquentes

### Pourquoi Mon état est désactivé

- Il faut avoir complété au moins un questionnaire pour avoir un statut.

### Pourquoi je vois une région par défaut sur Maps

- La permission de localisation a été refusée ou la position est indisponible.
- Activez la localisation dans les paramètres du téléphone, puis relancez l'onglet Maps.
